Project Title: Designing an Integrated System for Bonus, Leave, and Inventory Management with Spring and PostgreSQL

Project Overview:
This internship project entails creating a robust schema to streamline the management of employee-related information, including bonuses, vacation requests, and inventory assignments. The primary goal is to develop a backend solution using the Spring framework and PostgreSQL database tailored to the organization's operational needs.

During this internship, you will gain practical experience with Java programming and engage in real-world scenarios, building a strong understanding of the software development lifecycle. To facilitate your work, a SQL script will be provided to establish a sample database along with another script to insert initial data for testing and development purposes.
